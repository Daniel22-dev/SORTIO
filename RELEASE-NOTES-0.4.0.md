# SORTIO 0.4.0 – balíček 4

## Nové

- bezpečný projekční režim;
- časovač a stopky;
- kostka, mince, náhodné číslo a rozhodovač;
- dobrovolník, jinak spravedlivá náhoda;
- týmové skóre;
- historie a pokrytí zapojování;
- spravedlivý výběr nejméně zapojených;
- tisk skupin, míst, zapojení a kartiček;
- PDF přes tiskový dialog;
- datový model v4 a migrace z v2/v3.

## Ochrana dat

Projekce je oddělena od interních profilů a pravidel. E-mailové adresy se nadále neukládají. Tisk probíhá lokálně bez serverového generování.

## Opravy při testování

- předvolený časovač se už při spuštění nepřepisuje vlastní výchozí hodnotou;
- souhrn nastavení započítává historii zapojování;
- mobilní navigace počítá s modulem Nástroje.
